== Description ==

Praktická kalkulačka čisté mzdy od Měšec.cz vám během chvíle spočítá výši vaší čisté mzdy. Jednoduchý a rychlý způsob, jak si spočítat, kolik peněz se vám objeví na výplatní pásce. Zadejte jednoduše do požadovaného pole typ pracovního vztahu, hrubou mzdu, slevy na dani a výsledky jsou na světě. Plugin obsahuje odkaz na poskytovatele kalkulačky.

Kalkulačku vložíte do stránky pomocí značky `[kalkulacka-ciste-mzdy]`

== Installation ==
1. V administraci na stránce `Pluginy » Instalace pluginů` vyhledejte
kalkulačku a nainstalujte ji, nebo ji stáhněte z webu, rozbalte a nahrajte složku `kalkulacka-ciste-mzdy` do složky `/wp-content/plugins/`.
2. V administraci na stránce `Pluginy » Přehled pluginů` aktivujte plugin "Kalkulačka čisté mzdy".
3. Použijte stránku `Nastavení » Kalkulačka čisté mzdy` pro nastavení pluginu.
